# __게임 프로그래밍(기말) 기획서__

### __원본 게임__: A Simple 2D Platformer!
> 원본게임에 기능을 추가 할 예정.<br/>
> 원본 소스코드: https://www.youtube.com/watch?v=12d4WtmrEZs

### __프로젝트 마감일__: 2019/12/18

### __사용 엔진__: 유니티 + C#

### __추가 및 개선, 변경된 기능__
  * Tilemap 시스템으로 변경
  * Platform 이미지 변경
  * 카메라가 플레이어를 따르도록 변경
    * Smooth Camera follow 추가
  * 플레이어 이미지 변경
    * Idle 애니메이션 추가
    * Running 애니메이션 추가
    * Jumping  애니메이션 추가
    * Crouching  애니메이션 추가
  * 플레이어 컨트롤 로직 개선
    * 비선형적인 플레이어 이동로직으로 개선
    * 비선형적인 플레이어 멈춤로직 추가
    * 떨어지는 플랫폼상 점프기능 후가
